🌐 ¿Qué es GitHub?

GitHub es una plataforma en la nube que permite almacenar proyectos y gestionarlos con Git. Facilita el trabajo colaborativo, el control de versiones y la publicación de proyectos de software. Es esencial para compartir código con otros desarrolladores y trabajar en equipo de forma organizada.

🌍 Ir a GitHub

⚙️ ¿Qué es Git?

Git es un sistema de control de versiones distribuido, creado por Linus Torvalds. Permite gestionar los cambios en el código fuente, trabajar en múltiples ramas y mantener un historial completo de modificaciones. Es la base que hace posible el funcionamiento de GitHub.

🚀 Guía rápida de inicio

  • Instalar Git en tu PC (Descargar aquí).
  • Configurar usuario: git config --global user.name "Tu Nombre"
  • Configurar email: git config --global user.email "tuemail@ejemplo.com"
  • Inicializar repositorio: git init
  • Agregar archivos: git add .
  • Guardar cambios: git commit -m "Primer commit"
  • Conectar a GitHub: git remote add origin URL
  • Subir código: git push -u origin main
